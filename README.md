A modern Next.js project.

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

Using **npm**:

```bash
npm install
```

Or using **yarn**:

```bash
yarn install
```

### 3. Run the development server

Using **npm**:

```bash
npm run dev
```

Or using **yarn**:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 📦 Build for production

Using **npm**:

```bash
npm run build
npm start
```

Or using **yarn**:

```bash
yarn build
yarn start
```

## 🧪 Linting

Using **npm**:

```bash
npm run lint
```

Or using **yarn**:

```bash
yarn lint
```
